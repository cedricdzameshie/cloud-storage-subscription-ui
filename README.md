# Cloud Storage Subscription UI

A SaaS-style frontend application that simulates cloud storage plan selection and a subscription onboarding flow.

This project was built to practice component-based UI development, user-focused workflows, and responsive design patterns commonly found in real-world web applications.

---

## 🚀 Features
- Cloud storage plan selection and comparison
- Simulated subscription onboarding flow
- Responsive layout for desktop and mobile
- Basic form validation and user feedback
- Clean, reusable UI components
